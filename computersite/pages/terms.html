  • 1.   
    COMPUTER
    : A programmable, electronic device that accepts data input, performs processing operations on that data, and outputs and stores the results.

  • 2.   
    INPUT
    : The process of entering data into a computer; can also refer to the data itself.

  • 3.   
    PROCESSING
    : Performing operations on data that has been input into a computer to convert that input to output.

  • 4.   
    OUTPUT
    : The process of presenting the results of processing; can also refer to the results themselves.

  • 5.    
    STORAGE
    : The operation of saving data, programs, or output for future use.

  • 6.   
    COMMUNICATIONS
    : The transmission of data from one device to another.

  • 7.   
    DATA
    : Raw, unorganized facts.

  • 8.   
    INFORMATION
    : Data that has been processed into a meaningful form.

  • 9.   
    HARDWARE
    : The physical parts of a computer system, such as the keyboard, monitor, printer, and so forth.

  • 10.   
    DVD DRIVE
    : Reads and writes CD and DVD discs.

  • 11.   
    HARD DRIVE
    : Located inside the system unit; stores programs and most data.

  • 12.   
    PRINTER
    : Produces printed copies of computer output.

  • 13.   
    ROUTER
    : Connects devices so they can share an Internet connection and data.

  • 14.   
    MODEM
    : Connects the computer to the Internet.

  • 15.   
    USB FLASH DRIVE
    : Used to store documents, digital photos, music files, and other content to be moved from one PC to another.

  • 16.   
    SYSTEM UNIT
    : Case that contains the CPU, memory, power supply, storage device.

  • 17.   
    SPEAKERS
    : Produce audio output.

  • 18.   
    USB PORTS
    : Connect external devices that use the USB interface.

  • 19.   
    KEYBOARD
    : Used to type instructions into the computer; a primary input device.

  • 20.   
    MOUSE
    : Used to make on- screen selections; a primary input device.
